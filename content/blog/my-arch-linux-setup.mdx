# My Arch Linux Setup


I've been on Arch for about two years now. Here's my current setup.

## The stack

- **WM**: Hyprland — dynamic tiling with smooth animations
- **Bar**: Waybar with custom modules for brightness, volume, and system stats
- **Terminal**: Kitty with JetBrains Mono (same font as this blog)
- **Editor**: Neovim with LSP, Treesitter, and a hand-picked plugin set
- **Shell**: Zsh with zinit for plugin management

## Dotfile philosophy

Every config file has inline comments explaining what each option does. No `# TODO` — if an option isn't clear, I write a one-liner next to it.

I use GNU Stow to manage everything. A single `stow .` in the repo root creates all symlinks. No install scripts, no ansible, no friction.

## Why not NixOS?

NixOS is great, but I like knowing exactly what's on my system. Arch's minimal install + manual configs means I can rebuild from memory if I have to.
